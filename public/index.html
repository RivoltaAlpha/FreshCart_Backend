Welcome to FreshCart Backend API

This is the backend API for the FreshCart platform.
Manage products, categories, orders, stores, users, deliveries, payments, and more.
Explore the available endpoints and user roles below.

View Swagger API Docs
Admin
  • Full access to all resources
  • Manage users, stores, products, categories
  • View and manage all orders and deliveries
  • Access analytics and reports
  • Verify and moderate stores
Store Owner
  • Manage own store profile and products
  • View and process orders for their store
  • Track store analytics and revenue
  • Respond to customer feedback
Customer
  • Browse and search products
  • Place and track orders
  • Submit feedback and rate stores/products
  • Manage personal profile and addresses
Driver
  • View assigned deliveries
  • Update delivery status
  • Access delivery directions
